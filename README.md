# Recherche Sacrée — Atelier Gématrie · Isopséphie · Primosophie

Application web autonome de calcul **multi-systèmes** de la valeur numérique des mots, avec **accumulation locale** des entrées et **détection automatique des croisements** (mots de même valeur).

## 🔗 Utilisation

- **En ligne** : voir l'onglet **GitHub Pages** du dépôt (https://architechfr.github.io/recherche-sacree/).
- **Hors-ligne** : téléchargez `index.html` et ouvrez-le dans un navigateur. Tout fonctionne sans serveur ; les données sont stockées dans le navigateur (localStorage) et exportables en JSON.

## Systèmes de calcul

| Famille | Méthodes |
|---|---|
| **Latin / Français** | Ordinal (A=1…Z=26), réduction pythagoricienne |
| **Primosophie** (Vermard / Laveau) | Lettre → nombre premier — *table éditable* |
| **Hébreu** | Mispar Hechrachi, Gadol, Siduri, Katan, Katan Mispari, AtBash, Kolel |
| **Grec** | Isopséphie (système milésien) |

## ⚠️ Note sur la primosophie

La table officielle de Mathieu Laveau n'est pas publiée en ligne (réservée à son livre, 2024). La table fournie ici est **rétro-déduite** (A=1 puis suite des nombres premiers) et vérifiée sur les exemples connus `bien = mal = 69`. Elle est **éditable** dans l'onglet *Tables & réglages* — à confronter au livre.

Méthode complète, sources et précautions (biais de confirmation) dans [`RAPPORT-RECHERCHE.md`](RAPPORT-RECHERCHE.md).
